# Mini Projects with Symfony

## Description

We are making two app with symfony.

## Installation

```shell
git clone https://github.com/Bishwajitkuat/mini_project_symfony.git
cd mini_project_symfony
cp .env.example .env && cp web/.env.example web/.env
docker compose up -d
cd web/
composer install
```

- PHP
- Symfony
- Twig
